## Current Builds

### Information products and tools

- Idea Distillery - local call capture and planning output for turning messy project context into implementation-ready notes.
- Marble Melody - a private Next.js app that turns an uploaded MP3 into a browser-based rolling marble note machine.

### Trading, crypto, and market systems

- Daedalus Labs / Janus - Rust backtesting, forward-validation research, strategy sweeps, local service APIs, and reproducible artifacts.

- Older public trading experiments include [defi-dolphin](https://github.com/jstEagle/defi-dolphin), [staratlas-mm-bot](https://github.com/jstEagle/staratlas-mm-bot), and [sol-trading-bot](https://github.com/jstEagle/sol-trading-bot).

## What I Reach For
